# Irene working forms register

The following controls open live fictional-data workflows inside the demonstration:

1. Emergency Cover Activation
2. Return-from-Leave Briefing and Handover
3. Staff Availability / Leave Change
4. Break, Admin Buffer, Debrief and Support
5. Client Contact Outcome and Next Plan
6. Reception Message / Client Notification
7. Immediate Safety Concern
8. Critical Incident / Near Miss
9. Opening / Midday / End-of-Day / Practice Closure
10. Referral and Intake Evidence
11. Worker Event / WorkCover Review
12. Audit Export and Backup Snapshot

Each significant item retains a named owner, backup, due time, evidence requirement, escalation path and authorised closure step.
